# Indra interview backend
## Description

backend developer interview with nestjs, express and serverless

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run build && serverless offline

# production
$ npm run build && serverless deploy
```

## Test

```bash
# unit tests
$ npm run test
```
